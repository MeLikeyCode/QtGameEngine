Detailed Description

Represents a map which can contain a bunch of interacting Entities.

A Map has a PathingMap which keeps track of which cells are free and which are blocked. Each Entity can also contain its own PathingMap therefore the Map is notified every time an Entity is added/moved so that the Map's PathingMap can be updated accordingly.

A Map can be visualized by a Game. If the Map is currently being visualized by a game the game() function will return that Game, otherwise it will return nullptr.
