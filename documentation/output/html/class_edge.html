Detailed Description

Represents an edge between two Nodes.

Author
Abdullah Aghazadah
Date
4-21-15

An Edge is defined by the value of its starting ("from") and ending ("to") Nodes. Therefore two Edges are equivalent if they both start and end at the same (i.e. equivalent) Nodes.
