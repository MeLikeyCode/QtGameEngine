Detailed Description

Represents a sprite sheet.

A sprite sheet is a sheet of paper with tiles arranged in a grid like fashion. This is a value class.

Author
Abdullah Aghazadah
Date
3/18/17

Example usage:

SpriteSheet spriteSheet("path/to/file.png", 4,5,64,64); // the sprite sheet contains 64x64 tiles, 4 rows, and 5 columns
