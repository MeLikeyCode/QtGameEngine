Detailed Description

Represents a Gui that visualizes and allows mouse interaction with an Item.

When the Item in the ItemCell is clicked, ItemCell will emit a signal. You can set the Item of the ItemCell via setItem(Item*) or by passing the Item in the ItemCell constructor. There are several member functions provided which allow you to modify the look/size of the ItemCell.
