Detailed Description

A Mover that moves the Entity in a path finding way.

The entity will move using a path finding algorithm. You can specifiy if the entity should continue to face its target position or if it should face the direction its heading into.

Example usage:

PathMover* pm = new PathMover(entity);
pm->setAlwaysFaceTargetPosition();
pm->moveEntity(toSomePoint);
connect(pm,&PathMover::moved,this,myCallback);

◆ setStepSize()

void ECPathMover::setStepSize ( double  stepSize)

Sets how many pixels the entity should move every time he moves.

This in effect controlls the "granularity" of the movement. Higher values means the controlled entity takes bigger steps but infrequently. Lower values means the controlled entity takes frequent small steps. Note that this does not effect the speed of the controlled entity, just the movement "granularity"!
