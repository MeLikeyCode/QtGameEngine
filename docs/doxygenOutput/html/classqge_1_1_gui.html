Detailed Description

Abstract class that Represents a GUI element in a Game.

A Gui is placed relative to its parent Gui, if it does not have a parent Gui, its place relative to the top left hand corner of the Game's screen.

Most Guis offer functions for customizing their look in some way. Most Guis emit signals when they are interacted with. When a parent Gui is deleted, it will delete all of its child Guis.

virtual QGraphicsItem* qge::Gui::getGraphicsItem ( )

Returns the root QGraphicsItem that has the drawings for the Gui.

This is the QGraphicsItem that goes in the scene when Game::addGui() is called.

Implemented in qge::DialogGui, qge::ScrollWindow, qge::Panel, qge::QuestAcceptor, qge::ItemCell, qge::ShopGui, qge::AbilityCell, qge::ScrollBar, qge::QuestViewer, qge::InventoryViewer, qge::InventoryUser, qge::Button, qge::Label, and qge::Bar.

Returns the bounding box for the Gui.

The default implementation will return a bounding box that encompasses all the child Gui's bounding box. This function is virtual, so that sub classes may provide a better implementation if they wish.
