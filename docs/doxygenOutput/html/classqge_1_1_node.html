Detailed Description

Represents a node.

Author
Abdullah Aghazadah
Date
4-20-15

A Node simply represents a certain location using an x and y value. A Node object is defined by its x and y values. Two Nodes are equivalent if they have the same x and y values. A Node object is similar to a point except that a Node object is more discreet because its x and y values have to be integers.
