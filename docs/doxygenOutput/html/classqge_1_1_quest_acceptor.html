Detailed Description

Represents a Gui that allows you to select/accept quests.

Similar to a QuestViewer, the user is allowed to browse through some quests, however the QuestAccepter allows you to "accept" a quest by clicking its "Accept Selected Quest" button. Each time a Quest is accepted, a signal is emitted telling you which quest was just accepted. QuestAcceptor also has a close button, which simply closes the Gui (i.e. removes it from the game).

The recommended way to use the QuestAcceptor is:

  • create some Quest objects, put them in a Quests object
  • call QuestAcceptor::setQuests() passing in the quests object
  • listen to the quest accept signals to be notified of which quests the user accepts
