Detailed Description

A map controller that will spawn entities randomly within some region of a map.

The EntityCreator of the MCSpawner specifies what entity will be created. When it is time to create an entity, MCSpawner will get an entity from its EntityCreator, and then place that entity in a random spot in the map.
