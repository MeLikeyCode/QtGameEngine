int  fpsToPlayAt,
int  startingFrameNumber 
)
overridevirtual

Plays the specified animation.

Throws (assertion error) if the specified animation does not exist. If the specified animation is already playing (at current angle), does nothing (so if you want to restart, call stop() then play() again). If any other animation is currently playing, it will be stop()ed first.

Implements qge::EntitySprite.
